# BingWallpaper
### 需求
（url）http://www.bing.com/HPImageArchive.aspx?format=js&idx=0&n=1&mkt=en-US  
通过如上url获取到一个json对象  
（json）
```javascript

        {
            "startdate": "20160329",
```  

将url中的idx后面的参数由0进行++操作，下载下一张图片
